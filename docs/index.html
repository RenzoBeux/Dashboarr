Dashboarr

A mobile app to manage your self-hosted media stack from a single interface.

Google Play GitHub
Expo SDK 54 React Native TypeScript Open Source

Features

Unified Dashboard

All your services at a glance with customizable, reorderable cards.

Auto Network Switching

Detects your home WiFi and switches between local and remote URLs automatically.

No Backend Required

Connects directly to your services. Your data stays between your phone and your servers.

Push Notifications

Optional self-hosted backend for real-time notifications when downloads finish or media arrives.

Secure Storage

API keys stored in your device's secure enclave. No cloud, no accounts, no tracking.

Config Backup

Export and import your entire configuration with biometric authentication.

Supported Services

qBittorrent

Queue, pause, resume, delete, speed stats

Radarr

Search, add movies, monitor, queue

Sonarr

Search, add shows, episode monitoring

Overseerr

Browse, request, approve media

Tautulli

Active streams, bandwidth, history

Prowlarr

Indexer management, search, stats

Plex

Now playing, recently added, on deck

Glances

Server CPU, RAM, disk, network stats
